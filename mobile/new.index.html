Rutgers Mobile App

Stay connected with the new Rutgers mobile application, available for the iPhone, iPod, and iPad as well as Android phones and tablets. Access the University on the go: check campus bus times, dining hall menus, myRutgers, and more. Customize channels and visit Rutgers mobile web sites that work great whether you're on cellular or RUWireless.

Version History

Version 3.5 January 2, 2014

  • New Places: New channel lets you search for buildings, find the closest bus stop, and more
  • Emergency Services: Major channel redesign

Version 3.1 September 18, 2013

  • Fixed Updated display in the Schedule of Classes channel

Version 3.0 August 2, 2013

  • New Schedule of Classes: New channel includes classes for all Rutgers legacy campuses, for undergraduate and graduate level courses
  • Enhanced channel artwork
  • Infrastructure improvements: Better statistics and technology behind-the-scenes

Version 2.5 February 14, 2013

  • New RU-info: New channel for general Rutgers inquiries via SMS, email or voice
  • New NB Recreation: New channel providing hours of operation for recreation facilities
  • New NB Housing & ResLife: New Rutgers Mobile Web entry with features such as online maintenance request
  • New NB Kiosk: Touch-friendly Rutgers Mobile Web entry, as seen in the new Barnes & Noble bookstore
  • NB Career Services: Channel has updated content
  • R Alumni (Events): Additional event coverage

Version 2.2 November 27, 2012

  • Reduces application restarts
  • Increased reliability
  • Support for landscape display on more devices

Version 2.0 August 9, 2012

  • NewSupport for Android devices
