OSS Blog 2.0

Nov 5th, 2012 | Comments

We’re back, now with a shiny new blog powered by Octopress.

While waiting for hurricane Sandy to knock the power out last week, I got around to looking at some of the code I had written for the last version of this blog and realized it really wasn’t very good. I had also just recently deployed my own blog with Octopress, and learned enough sass that I figured I could restyle it to look almost exactly like the old website. It turned out to be a bit more complicated than I expected ( I didn’t realize how it resized things as you resize the window ), but I mananged to get it all working just in time for the power to go out.
