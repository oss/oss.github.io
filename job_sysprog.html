Now Hiring: Student Developers, Sysadmins, and Hackers

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

If you enjoy working on open source software or systems administration, Open System Solutions (OSS) would like to have a word with you.

Dependable self-starters are sought for an opportunity to work with the Enterprise Application Services: Enterprise Messaging group in Rutgers University's Office of Information Technology.

Responsibities:

  • Work in a team environment to design and develop open source software, web applications, scripts, etc. used by members of the Rutgers community
  • Configure and deploy open source software for use by the Rutgers community
  • Configure and maintain Linux servers
  • Build RPM software packages for the Rutgers community, when needed

Requirements:

  • Currently a student with one or more years left at Rutgers
  • Ability to work 12-20 hrs/wk on average during the semester
  • Ability to work 20-30 hrs/wk during the Summer of 2019.
  • Experience using the Linux/Unix command line.
  • Willingness to work in a Linux-only environment
  • Programming ability in at least one language
    • CS111 or CS112 homework does not impress us
  • Ability to lift 30lbs

Experience using any of the following is a plus, but not required:

  • Databases (MySQL, MongoDB, PostgreSQL, etc)
  • Version control systems (Git, SVN, Mercurial, etc)
  • Backend web application frameworks in any language (Flask, Django, Sinatra, Ruby on Rails, etc)
  • Software containers such as Docker
  • Server provisioning software (Chef, Puppet, Ansible, Salt, etc) a BIG plus

Pay is based on experience and ability. Flexible hours. Training provided.

To apply, please send a plain text or PDF copy of your resume with two references to oss@oss.rutgers.edu

Resumes sent in any other format besides plain text or PDF will not be considered. Anything resembling .doc or .docx will be laughed at and shredded.
